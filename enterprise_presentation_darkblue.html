Technical Architecture

Layer Technology Details
Frontend Azure App Service Python 3.11 on Linux, deployed from GitHub
Orchestration Promptflow Serverless, multi-step LLM pipeline
Model Azure OpenAI GPT-4o Chat model with temp=0, API key auth
Config Environment Variables PROMPTFLOW_ENDPOINT, API_KEY in App Service
Hosting Azure PaaS 100% Azure-native, Free tier (F1) available
